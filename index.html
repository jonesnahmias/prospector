Opção A — Upload manual de CSV ou XLSX

Importe CSV, TXT, XLS ou XLSX. Para arquivos Excel, o app lê a primeira planilha com dados.
Colunas aceitas: codigo, descricao, unidade, custo. Para insumos: classificacao, codigo, descricao, unidade, origem, custo. Opcionais para analítico: categoria, insumo_codigo, insumo_descricao, insumo_unidade, coeficiente, preco_unitario. Para SICRO, use o relatório analítico de composições em XLSX.

Opção B — Baixar base oficial

ORSE pode ser buscado pelo site oficial sem instalar o aplicativo. Para evitar travamentos, use um termo de busca e importe por lotes.

Opção C — Importar Analítico Individual

Use esta opção para a aba Analítico com Custo do SINAPI. No Excel, selecione a tabela azul do analítico, incluindo o cabeçalho Tipo Item, Código do Item, Descrição, Unidade, Coeficiente, Custo Unit., Custo Total, %AS, Situação, copie e cole abaixo.

Duplicidade: se já existir composição com o mesmo código, ao confirmar o app perguntará se deseja atualizar a existente ou criar uma nova versão analítica.
1. No ExcelSelecione da linha do cabeçalho azul até o último item da composição.
2. CopieUse Ctrl+C. O Excel copia como texto separado por tabulação.
3. Cole no appClique em “Pré-visualizar analítico” e confirme a importação.
Classificação automática dos itens: itens marcados como COMPOSICAO entram como Composição auxiliar. Itens marcados como INSUMO procuram a categoria no cadastro de insumos importado; se não encontrar, o app tenta identificar Material, Mão de obra ou Equipamento pela descrição.
